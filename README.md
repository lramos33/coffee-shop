## 💻 Instalando o projeto

Clone o repositório:

```
git clone git@github.com:lramos33/coffee-shop.git
```

Acesse a pasta do repositório

```
cd coffee-shop
```

Instale as dependências:
```
npm install
```

Execute a aplicação
```
npm run dev
```
